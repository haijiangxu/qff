QFF是基于Python语言开发的一个量化金融框架库，提供数据获取-数据存储-策略编写-策略分析-策略回测-模拟交易等一站式解决方案。

QFF为个人用户提供本地的回测和模拟交易环境，使用户更加专注于交易策略的编写。

.. note::

   请访问 `Github`_ 网站获取QFF源代码，如果有任何问题请访问 `Github Issues`_ 查询或提交。
