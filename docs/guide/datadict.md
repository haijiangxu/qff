# 数据字典

## 市值数据
每日更新

通过函数 {func}`.get_valuation` 和 {func}`.query_valuation` 获取

数据格式如下：
```{eval-rst} 

            ==========================  ====================
             字段名	                        含义
            ==========================  ====================
             code                         股票代码
             date                         日期
             quantity_ratio               量比
             capitalization               总股本(万股)
             circulating_cap              流通股本(万股)
             market_cap                   总市值(亿元)
             circulating_market_cap       流通市值(亿元)
             turnover_ratio               换手率(%)
             pe_ratio                     市盈率(PE, TTM)
             pe_ratio_lyr                 市盈率(PE)s
             pe_ratio_dyn                 市盈率（动态）
             pb_ratio                     市净率(PB)
            ==========================  ====================
```

## 融资融券数据
每日更新

通过函数 {func}`.get_mtss` 获取

数据格式如下：
```{eval-rst} 

        ==================  ====================
        字段名	                含义
        ==================  ====================
        date	            日期
        sec_code	        股票代码
        fin_value	        融资余额(元）
        fin_buy_value     	融资买入额（元）
        fin_refund_value	融资偿还额（元）
        sec_value	        融券余量（股）
        sec_sell_value	    融券卖出量（股）
        sec_refund_value	融券偿还股（股）
        ==================  ====================
```
